Classification from Scratch

Bizovi Mihai

July 16, 2018

https://freakonometrics.hypotheses.org/ has a lot of great content which is theoretically well founded. Carpentiers series on classification from Scratch drew my attention as an excellent way to go back to the basics of well known models and algorithms. Moreover, he provides minimalist R code which gets to the essence and can be improved as an exercise, which will be used for practice.

library(tidyverse)
library(ggthemes)
library(GGally)
data <- data.frame(
  x1 = c(.4, .55, .65, .9, .1, .35, .5, .15, .2, .85),
  x2 = c(.85, .95, .8, .87, .5, .55, .5, .2, .1, .3),
  y  = c(1, 1, 1, 1, 1, 0, 0, 1, 0, 0)
)

p <- data %>% 
  mutate(y = as.factor(y)) %>%
  ggplot(aes(x1, x2, color = y)) + 
  geom_point(size = 3) + 
  theme_minimal() + 
  labs(x = "First Feature", y = "Second Feature") + 
  guides(color = guide_legend(title = "target")) + 
  scale_color_ptol()
p
